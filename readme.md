PROBLMEA:

```
Create un array di oggetti ( con proprietà text e done )che rappresentano le task e mostratele a schermo. 
Aggiungete poi le funzionalità di cambio dello stato del task (cambiate il valore di done), eliminazione di una task ed aggiunta di una nuova task.

```

SOLUZIONE: 

```
1.  Crea un array di oggetti: Inizializza un array vuoto che conterrà gli oggetti delle task. 
    Ogni oggetto rappresenterà una task e avrà due proprietà: "text" per il testo della task e "done" per lo stato della task.

2.  Mostra le task a schermo: Scansiona l'array di oggetti delle task e mostra a schermo il testo di ogni task insieme al suo stato. 
    Utilizzare un ciclo per iterare attraverso l'array e stampare il testo e lo stato di ogni task su una nuova riga.

3.  Cambia lo stato della task: Aggiungi la funzionalità per cambiare lo stato di una task. 
    3.1 Consentire all'utente di fare clic su una task per cambiarne lo stato da "non completata" a "completata" e viceversa. 
    3.2 Quando viene cambiato lo stato di una task, aggiorna la visualizzazione a schermo per riflettere il nuovo stato.

4. Elimina una task: Aggiungi la funzionalità per eliminare una task.
    4.1 Consentire all'utente di fare clic su un pulsante o un'icona accanto a una task per rimuoverla dall'array. 
    4.2 Una volta eliminata la task, aggiorna la visualizzazione a schermo per rimuoverla dalla lista.

5. Aggiungi una nuova task: Aggiungi la funzionalità per aggiungere una nuova task all'array di oggetti.
    5.1 Puoi fornire un'interfaccia in cui l'utente può inserire il testo della nuova task e premere un pulsante per aggiungerla all'array.
    
```
